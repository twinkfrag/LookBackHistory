LookBackHistory
--

Mozilla Firefox および Google Chromeの履歴から検索します。
タイトル/アクセス日付範囲/URLから複合条件で検索することができます。

はじめにLoadボタンでFirefoxまたはChromeの履歴を読み込んでください。
